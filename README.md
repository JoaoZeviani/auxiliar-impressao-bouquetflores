# Auxiliar de Impressão - Bouquet Flores

Programa para preencher e imprimir os papéis físicos da Bouquet Flores:

- Pedido
- Cartão com dizeres
- Cartão sem dizeres

## Como usar

1. Preencha os dados do pedido e/ou cartão.
2. Use **Copiar destinatário** se quiser levar nome/endereço para o cartão.
3. Use **Preview** para conferir.
4. Use **Imprimir pedido** ou **Imprimir cartão**.

Na impressão, sai apenas o texto preenchido. O modelo do papel não é impresso.

## Configurações

Em **Configurações**, você pode:

- ajustar posição dos campos;
- escolher fonte dos dizeres;
- adicionar/remover vendedores;
- preencher um pedido teste;
- copiar posições para ajuste futuro.

## Celular

Abra o site no navegador e use **Baixar aplicativo** ou **Adicionar à tela inicial**.

## Impressão

Para encaixar corretamente no papel físico, use escala **100% / tamanho real**.
