# Auxiliar de Impressão - Bouquet Flores

Versão v8.

## GitHub Pages

Para publicar no GitHub Pages, envie somente os arquivos do programa web:

- index.html
- styles.css
- app.js
- manifest.webmanifest
- service-worker.js
- assets/

Não envie a pasta dados, arquivos .bat, node_modules ou package.json.

## Alterações da v8

- Botão de instalação muda conforme o sistema:
  - Android/iPhone: Baixar aplicativo.
  - Windows: Adicionar atalho à área de trabalho.
- O botão de instalação/atalho não aparece quando o programa já foi aberto como aplicativo/atalho PWA.
- Preview do pedido no celular agora é reduzido automaticamente para caber inteiro na tela.
- Cache do PWA atualizado para v8.

## Impressão

Na impressão, o modelo do papel não é impresso. Saem somente os dados preenchidos. Para melhor fidelidade, use escala 100% / tamanho real.
